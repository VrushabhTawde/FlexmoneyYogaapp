# FlexMoney
Full stack yoga registration


## Run Locally

Clone the project

```bash
  git clone https://github.com/gauravmandal1/FlexMoney.git
```

Go to the FlexMoney directory

```bash
  cd FlexMoney
```

Install dependencies of backend

```bash
  cd server
  npm install i
```

Start the server

```bash
  npm start
```
Install dependencies of frontend

```bash
  cd client
  npm install i
```

Start the server

```bash
  npm start
```


## Tech Stack

**Client:** React,CSS

**Server:** Node, Express , MYSQL
